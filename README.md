# India-Clock-Screensaver
Windows Screensaver
## Installation
Download and run `INDIA_Clock_Screensaver_Setup.exe`.

This installer installs the screensaver for the current user only.
Administrator privileges are not required.

## Security
The installer has been scanned with multiple antivirus engines.
All major antivirus products report it as clean.

SHA-256 checksum is provided for integrity verification.
